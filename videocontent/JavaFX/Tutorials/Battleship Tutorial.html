Multiplayer Battleship Tutorial

This is an ongoing video series, of a currently unknown number of episodes.
The idea is to create a multiplayer battleship game. We'll use JavaFX, the MVVM template structure, and sockets.
Eventually there will be the functionality of

  • Logging in
  • Creating a new user
  • Some kind of game lobby
  • Playing an actual game against an opponent
